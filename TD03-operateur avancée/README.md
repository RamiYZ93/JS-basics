 TD3 – Utilisation avancée des opérateurs en JavaScript
 Description

Ce projet contient une série d’exercices portant sur l’utilisation avancée des opérateurs en JavaScript :

Opérateurs logiques (&&, ||)

Opérateur modulo (%)

Opérateur d’addition (+)

Concaténation de chaînes

Opérateur d’incrémentation par accumulation (+=)

L’objectif est de comprendre le fonctionnement des opérateurs dans des situations concrètes (jeu vidéo, test de nombres pairs, conditions logiques…).
