# TD01 - Variables

## Objective

Practice declaring variables and displaying values using `console.log()` in JavaScript.

---

## Instructions

1. Create a variable representing a dog and assign it a name.
2. Create a variable representing a cat and assign it a name.
3. Display both variables in the console.
4. Print a sentence introducing the dog's name.
5. Print a sentence introducing the cat's name.
6. Create a variable representing the sound made by the cat.
7. Create a variable representing the sound made by the dog.
8. Display sentences combining the animal name and its sound.
9. Modify the value of the cat's sound.
10. Display a final sentence showing the updated sound.

---

## Concepts Practiced

- Variable declaration (`let` / `const`)
- String values
- String concatenation
- Console output
- Variable reassignment

---

## Goal

Understand how to declare, use, and update variables in JavaScript.
